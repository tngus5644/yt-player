# 출시 노트 (Release Notes)

Play Console "출시 노트" 입력란은 **언어별 최대 500자**입니다. 아래 버전 1.0.0 최초 출시용 초안과 함께 후속 업데이트 시 재사용 가능한 템플릿을 포함했습니다.

## Play Console 입력 팁

- **언어별 등록**: `ko-KR` 필수, `en-US`는 선택이지만 검색 노출에 유리합니다.
- **길이 제한**: 500자 초과 시 자동 컷 — 핵심 문구를 앞 250자에 배치하세요.
- **이모지 사용**: 1~3개까지는 시각적 포인트에 효과적, 과하면 스팸처럼 보임.
- **매 업데이트마다** 출시 노트를 작성하는 것이 리텐션 지표에 유리합니다.
- **버그 수정만**인 경우에도 "안정성 개선" 한 줄이라도 적는 것이 좋습니다 (빈 노트 피하기).
